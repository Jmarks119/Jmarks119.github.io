Pizza Menu Javascript Page

This served as a basic runthrow to make sure we understood the essentials of selecting elements, binding event listeners, as well as writing functions in a JavaScript page. For my solution, I wrote a class for the receipt with a property for each menu section

Girls Frontline Logistics Calculator

The community of the mobile game Girls Frontline is currently using plain spreadsheets hosted online as a resource for players to decide which logistics missions will give them the most in-game resources over time. The first goal of this project was moving this data from a general use spreadsheet to a responsive webpage more specifically designed to present the data, in a responsive table that will become more compact on smaller screens. The next goal is to unobtrusively add JavaScript to the page in order to provide enhanced functionality, providing users the means to generate weighted results, filter missions, and calculate an overall total resource gain.

The Syndicate WebApp

An ambitious project to provide the community of the online game Final Fantasy XIV with a new tool for analyzing and improving decisions they make related to the in-game economy. The Django WebApp will be built around a database that will store records of market transactions and present that information to users along with some functionality in the areas of data analytics for deriving useful information from those collected transaction records. The project also involves a small c# application that will be distributable to users for the purpose of collecting this market information from their own game-servers for uploading to the WebApp.
